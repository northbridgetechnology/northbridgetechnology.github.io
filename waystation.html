Everything Your Team Needs

From ticket creation to resolution analytics, Waystation covers the full support lifecycle.

Guided Setup Wizard

A 12-step first-run wizard walks admins through every configuration — admin account, projects, ticket types, custom fields, mail, SSO, storage, and backup — before any agent logs in.

Ticket Management

Per-project numbering, nine statuses, four priority levels, custom ticket types with colors and icons, parent/child relationships, bulk operations, and ticket merging with full customer redirect and audit trail.

Customer Portal

Customers submit tickets through branded forms — no account required. Magic-link access lets them track progress, reply, and upload attachments.

Approval Workflows

Multi-round approvals with "any-one" or "all-required" modes. External approvers respond via magic link — no login needed.

Kanban & Sprints

Drag-and-drop Kanban boards with WIP limits and sprint management with story-point tracking. Create cross-project boards to manage tickets from multiple projects in a single unified view.

SLA Tracking

Define per-project, per-priority response and resolution targets. Business-hours-aware calculations with first-touch and first-response timestamp tracking, holiday exclusions, and breach-triggered automation.

Automation Engine

Build rules with eight trigger types, flexible AND/OR conditions, and actions like status changes, assignments, tagging, and email notifications. Mark rules as default to auto-apply them across all projects.

Reporting & Analytics

Custom reports with volume charts, SLA compliance metrics, resolution-time distributions, and agent performance. Export to XLSX or PDF. Share reports directly with customers via a secure link.

Email Integration

Ingest tickets from Office 365 and Exchange via Microsoft Graph. Threaded replies are matched to existing tickets automatically.

Identity & Access

Local authentication with JWT, Azure AD single sign-on, and SCIM 2.0 automated provisioning from Azure AD or Okta.

Targeted Comments

When multiple customers share a ticket, direct comments at all customers or a specific individual — keeping communications relevant and private where it matters.

Webhook Ingestion

Accept tickets from any external system via incoming webhooks. Use JMESPath field mapping to extract data from any payload shape — no code changes required.

Built Different

The features that set Waystation apart from SaaS ticketing tools.

Self-Hosted & Fully Yours

Deploy on your infrastructure in minutes with a single Docker Compose command. Simple, quick, and efficient — your data never leaves your network. No per-seat fees, ever.

Compliance-Ready Exports

Export tickets as ZIP bundles with PDF metadata and all attachments — ready for audits and regulatory review.

Magic-Link Approvals

External stakeholders approve requests via a secure email link. No accounts to manage, no friction.

Flexible Storage Backends

Store attachments locally or switch seamlessly to AWS S3, Azure Blob Storage, or Google Cloud Storage.

Multi-Project Architecture

Independent ticket numbering, custom statuses, ticket types, and SLA rules per project. Cross-project Kanban boards give agents a unified view across any combination of projects.

Built-In Backup & Restore

Scheduled PostgreSQL backups with optional upload to SharePoint. One-click browser-based restore when you need it.

Safe Ticket Moves

Moving a ticket between projects? Custom field data is preserved where names match. Unmapped fields become a documented internal comment — zero data loss.

IP-Level Audit Logging

Every admin action is logged with timestamp, user, and source IP address — giving you a forensic-grade audit trail for compliance and security reviews.
